The Lord of the Rings Tribute theme by MyWebResource.

Instructions:
-------------
Extract the folder "lotr_v2" and place it in your xoops/themes folder.  There are three header graphics included with this theme in addition to the rotating header graphic.  To change your header graphic, just edit the style.css file at the following line with the graphic name of your choice.
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
td#headerimg {width: 100%; background-color: #000000; vertical-align: middle; text-align:center; background-image: url(header.gif); background-position: right;}
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=

Enjoy!
